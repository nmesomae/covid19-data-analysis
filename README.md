# covid19-data-analysis
COVID-19 data analysis and visualization using Python

## Project Overview
This project analyzes global COVID-19 data to uncover trends in infection rates, recovery statistics, and mortality patterns using data analysis techniques.

## Objectives
- Clean and preprocess real-world datasets
- Perform exploratory data analysis (EDA)
- Visualize pandemic trends across countries

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified infection growth trends over time
- Compared mortality rates between countries
- Generated data visualizations for better interpretation
